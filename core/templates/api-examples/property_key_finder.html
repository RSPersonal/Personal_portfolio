Keyword finder

A tool to find certain keywords in .txt file. I wrote this tool because at some point I needed to find certain keywords in a (sometimes long) class file. Instead of manually looking through the whole code I wrote this script to find these keywords and write them out in txt file or show direct in browser.

Step by step guide.

  1. Open Notepad or any text file editor of your choice.
  2. Open your (class)file in an IDE of your choice, in which you want to find the keyword
  3. Copy the content from the file to the text file in Notepad and save the file. *Important to save the file as .txt
  4. Select desired output
  5. Upload the file down below
