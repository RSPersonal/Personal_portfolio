Welcome to the stocktracker

I build this stocktracker app to learn database operations. To get current stock data, I linked this to the free API from Polygon.io. It's only possible to do 5 API calls per minute. You will see an error message whenever the api calls are exceeded.

Current features are:

  1. Basic CRUD database operation
  2. Adding portfolio's linked to user
  3. Only show portfolio's linked to user
  4. Show general KPI's in portfolio overview
  5. KPI's calculated based on positions in portfolio
  6. Adding position to portfolio
  7. API connection with Polygon.io to get live stock data
  8. Chart.js integration for visual representation of portfolio metrics
