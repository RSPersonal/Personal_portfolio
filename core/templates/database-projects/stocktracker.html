Welcome to the stocktracker

I build this stocktracker app to learn database operations. To get current stock data, I linked this to the free API from Polygon.io. It's only possible to do 5 API calls per minute. You will see an error message whenever the api calls are exceeded.

Current features are:

  • Basic CRUD database operation
  • Adding portfolio's linked to user
  • Only show portfolio's linked to user
  • Show general KPI's in portfolio overview
  • KPI's calculated based on positions in portfolio
  • Adding position to portfolio
  • API connection with Polygon.io to get live stock data
